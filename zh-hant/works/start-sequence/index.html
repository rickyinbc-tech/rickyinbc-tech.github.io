作品・Motion

起跳序列

多名泳手的起跳被壓縮成跨越泳線的下降節奏。

作品資料

起跳序列

  • 英文標題： Start Sequence
  • 系列： Motion
  • 狀態： 收藏、展覽及授權狀態由工作室按作品書面確認。

泳手在空中與入水之間形成連續弧線，紅色泳線、碧綠水面、身體與水花共同組成一段視覺樂句。作品沒有孤立某位運動員，而是把競賽轉化為序列、編舞與重複；畫面由上而下推進，最後一個身體在水中化成泡沫。

圖說：多名泳手的起跳被壓縮成跨越泳線的下降節奏。
