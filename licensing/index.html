License Notes

Use requests should identify the image, format, territory, duration, and deadline.

This page describes the licensing reference used for image metadata across the official Ricky Kwok portfolio. It does not grant an automatic public license. It identifies the copyright holder and gives editors, curators, publishers, and institutions a clear route to request permission.

For image licensing, include the artwork title or page URL, intended use, publication or project name, media format, territory, duration, print run or audience size, and deadline. For fine-art print acquisition, include preferred print size, framing preference, delivery city, and collection or exhibition context.

Usage Information

Licenses are prepared for the specific approved use.

Displaying a photograph on this portfolio does not place it in the public domain. Copyright remains with Ricky Kwok, and reuse requires a written agreement.

Editorial and Publishing

State the publication, article or book context, language, territory, print run, digital audience, placement, dimensions, and publication date.

Exhibitions and Catalogues

Identify the venue, exhibition title, dates, display format, catalogue requirements, requested works, and whether files or physical prints are needed.

Institutional and Commercial

Describe the organization, campaign or installation, channels, territory, duration, audience, and any cropping, text overlay, or adaptation requested.

Credit and Files

Approved uses should carry the agreed credit line. File size, colour profile, delivery method, fee, and usage limits are confirmed with the license.
