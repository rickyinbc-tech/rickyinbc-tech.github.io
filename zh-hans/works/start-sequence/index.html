起跳序列

  • 英文标题： Start Sequence
  • 系列： Motion
  • 艺术家: Ricky Kwok / 郭文棣 / Kwok Man Tai
  • 作品类型: 摄影作品
  • 版权: © 郭文棣 Ricky Kwok。任何复制或使用均须事先取得书面许可。

泳手在空中与入水之间形成连续弧线，红色泳线、碧绿水面、身体与水花共同组成一段视觉乐句。作品没有孤立某位运动员，而是把竞赛转化为序列、编舞与重复；画面由上而下推进，最后一个身体在水中化成泡沫。

图说：多名泳手的起跳被压缩成跨越泳线的下降节奏。
