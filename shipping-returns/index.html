Collector information

Shipping, installation, and returns are confirmed before payment.

The studio provides a written offer for the specific artwork, size, destination, framing choice, production schedule, and delivery method.

Shipping quote

Shipping origin, destination eligibility, packaging, insurance, duties, taxes, and delivery timing are quoted with the work. The buyer should not assume that a preview price includes shipping or import charges.

Packaging and inspection

Works are packed for the agreed format. Please inspect the package on arrival, photograph visible damage before opening where possible, and contact the studio promptly with the lead ID and photographs.

Framing and installation

Framing, glazing, mounting, and installation are optional and must be stated in the written offer. Installation guidance can be supplied for the agreed size and display environment.

Cancellation and returns

Because many works are produced to the confirmed size or framing specification, cancellation and return rights depend on the written offer and applicable consumer law. No return term is implied until it is stated in writing.

Before you inquire

Tell the studio where the work is going and how it will be displayed.

Include the artwork title, preferred dimensions, framed or unframed preference, delivery city and country, intended setting, and any deadline. The studio will reply with the current edition and production facts available for that work.
